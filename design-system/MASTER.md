# Interview Copilot — Design System

> Generated per [UI UX Pro Max](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) — AI Copilot + Real-Time Monitoring

## Pattern
Bento dashboard — meeting zone | answer panel | transcript bar

## Style
AI-Native UI + Glassmorphism on Dark OLED

## Colors
| Token | Hex | Usage |
|-------|-----|-------|
| bg-base | #070b10 | Page background |
| bg-elevated | #0f1419 | Panels |
| bg-glass | rgba(15,20,25,0.72) | Glass surfaces |
| border | rgba(148,163,184,0.12) | Dividers |
| text-primary | #f1f5f9 | Headings, answers |
| text-muted | #94a3b8 | Labels |
| accent | #2dd4bf | Primary CTA, links (teal) |
| accent-hover | #14b8a6 | Hover |
| live | #f59e0b | Listening indicator |
| danger | #f87171 | Stop, errors |
| success | #34d399 | Configured state |

## Typography
- UI: DM Sans
- Answer body: Source Serif 4 (long-form readability)

## Effects
- backdrop-blur-xl on panels
- transition 200ms ease
- Soft shadow: 0 4px 24px rgba(0,0,0,0.4)
- No purple/pink AI gradients
